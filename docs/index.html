# features

☕

Caffeinate

Wraps macOS caffeinate -dis to prevent display, idle, and system sleep. Your screen stays on.

🔒

Lock All Input

System-wide CGEventTap intercepts and suppresses every keyboard, mouse, scroll, and trackpad event.

🔑

Secret Unlock Combo

Set your own key combination. Only you know it. Not shown on the lock screen.

⏰

Live Dashboard

Rich-powered lock screen with a real-time session timer. Looks clean, stays out of the way.

🚨

Panic Reset

Press Escape 5 times rapidly to force-unlock. Built-in, always works, can't be disabled.

💻

CLI Native

No GUI. Pure terminal tool. Install with Homebrew or pip, run from any shell.

# usage

# Lock with default combo (ctrl + cmd + u)
$ vegitate

# Custom unlock combo
$ vegitate -c ctrl+shift+q

# Allow mouse cursor movement (clicks still blocked)
$ vegitate --allow-mouse-move

# Skip caffeinate if running separately
$ vegitate --no-caffeinate

# how it works

1

Caffeinate

Starts caffeinate -dis to prevent display, idle, and system sleep.

2

Event Tap

Creates a CGEventTap at kCGSessionEventTap that intercepts all HID events before any application sees them. Returns None to suppress.

3

Unlock Detection

Your secret key combo is detected inside the tap callback itself. When matched, the tap is removed, caffeinate stops, and input is fully restored.

4

Auto Re-enable

If macOS disables the tap (timeout), it is automatically re-enabled. The lock never silently breaks.
